Dieser Server ist ein JonDonym Exit Mix

Dieser Server ist ein anonymer Proxy Server ohne Logdaten. In der Regel wird diese Webseite aufgerufen, weil sie Probleme bei der Nutzung ihres Angebotes durch diese IP-Adresse festgestellt haben. Dieser Server ist Teil einer JonDonym Kaskade. Der Service dient der Sicherung der Privatsphäre von ganz normalen Menschen. Dieser Router unter dieser IP-Adresse generiert keinen eigenen Traffic, sondern leitet nur Daten weiter.
JonDonym wird von vielen Menschen genutzt, einschließlich Whistleblowern, Journalisten, chinesischen Dissidenden, welche die Zensurin ihrem Land umgehen wollen, um nur einige Beispiele zu nennen. JonDonym wurde nicht entworfen, um Angriffe aus dem Internet zu unterstützen, der Dienst wird aber auch dafür genutzt. Die missbräuchliche Nutzung ist jedoch sehr gering, da Kriminelle und Hacker signifikant bessere Möglichkeiten haben, ihre Aktivitäten zu verschleiern. Nach unserer Ansicht überwiegt die Notwendigkeit unbeobachteter und unzensierter privater Kommunikation für viele Menschen den gelegentlichen Missbrauch durch einige Wenige.

JonDonym Mixe speichern keine Informationen über den weitergeleiteten Traffic. Im Falle strafrechtlich relevanten Missbrauchs können die Betreiber der Mix Server wenig zur Aufklärung der Straftat beitragen. Ermittler sind auf konventionelle Methoden der Strafverfolgung angewiesen.

Um zukünftigen Missbrauch des Dienstes gegen ihr Web-Angebot auszuschließen, besteht die Möglichkeit, diesen und weitere Server ihres Netzwerkes in die Empfehlung für die JonDonym Blockliste aufzunehmen.
